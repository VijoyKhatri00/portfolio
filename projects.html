Autonomous Drone Development

April 1, 2023 - April 19, 2023

Mentor: Prof. Atul | Team Size: 1 (Solo Project)

Project Overview

In this solo project, I developed a fully functional drone by designing, constructing, and testing its various components. The project involved setting up brushless motors, integrating flight control systems, and programming the drone for remote operation using Arduino.

Key Features

  • Custom-built drone frame and component assembly
  • Brushless motor integration for optimal performance
  • Arduino-based flight control system
  • Remote control operation with real-time response
  • Multiple sensor integration for flight stabilization
  • Safety systems and fail-safe mechanisms

Technical Implementation

The drone utilizes various sensors including gyroscopes, accelerometers, and barometric pressure sensors to maintain stable flight. The flight control algorithm processes sensor data in real-time to adjust motor speeds and maintain stability during flight operations.

Outcome

Successfully demonstrated the drone's ability to fly autonomously and respond to remote commands effectively. The project showcased practical application of IoT, sensor integration, and real-time control systems.

Alcohol Detection & Prevention System

April 1, 2022 - April 29, 2022

Team Size: 4

Project Overview

This project focused on developing a comprehensive system to detect alcohol levels in drivers and generate real-time alerts to improve road safety. The system integrates hardware sensors with intelligent software algorithms.

Key Features

  • MQ-3 alcohol sensor integration for precise detection
  • Real-time alcohol level monitoring
  • Automated alert system for safety violations
  • In-vehicle implementation ready design
  • Data logging and reporting capabilities
  • User-friendly interface for monitoring

Technical Implementation

The system uses the MQ-3 alcohol sensor connected to an Arduino microcontroller to detect the presence of alcohol in a driver's breath. When alcohol levels exceed permissible limits, the system triggers immediate alerts and can integrate with vehicle systems for safety measures.

Impact

The functional prototype successfully detected alcohol levels and triggered safety alerts, demonstrating potential for real-world implementation in vehicle safety systems.

SpaceX Falcon 9 Landing Prediction

IBM Applied Data Science Capstone Project

Project Overview

Applied Data Science project analyzing SpaceX Falcon 9 rocket launches to predict first stage landing success. The analysis helps determine launch costs for competitive bidding against SpaceX.

Business Problem

SpaceX advertises Falcon 9 rocket launches at $62 million, while other providers cost upward of $165 million. Much of the savings comes from SpaceX's ability to reuse the first stage. Predicting landing success helps determine actual launch costs.

Methodology

  • Data collection from SpaceX API and web scraping
  • Exploratory data analysis and visualization
  • Feature engineering and data preprocessing
  • Machine learning model development
  • Model evaluation and comparison
  • Interactive dashboard creation

Key Insights

Developed predictive models achieving high accuracy in determining first stage landing success, providing valuable insights for cost estimation and competitive analysis.

Technologies Used

Python Pandas NumPy Scikit-learn Matplotlib Seaborn Plotly Jupyter Notebook Machine Learning Data Visualization
